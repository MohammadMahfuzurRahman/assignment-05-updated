1️⃣ Difference between var, let, and const?
Ans:var, let, and const are used to declare variables in JavaScript, but they behave differently.
var:var is function-scoped and allows both redeclaration and reassignment, which can sometimes create confusion in larger programs. 
let:let  is block-scoped, meaning it only works inside a specific block, and it allows reassignment but not redeclaration.
const:const is also block-scoped, but its value cannot be changed after it is assigned, so it is mainly used for fixed values.

2️⃣ What is the spread operator (...)?
Ans:The spread operator is used to expand or unpack elements from arrays or objects. It helps to combine, copy, or manipulate data easily without modifying the original structure.It is commonly used to make code shorter and more readable.

3️⃣ Difference between map(), filter(), and forEach()?
Ans:These are array methods used for processing data.
map():map() is used when we want to transform each element and create a new array.
filter():filter()  is used to select elements based on a condition and also returns a new array.
forEach():forEach()is simply used to loop through elements and perform an action, but it does not return any new array.

4️⃣ What is an arrow function?
Ans:An arrow function is a shorter and more modern way of writing functions in JavaScript.It uses a special arrow syntax and helps make the code more concise and easier to read.It is widely used in modern JavaScript, especially when working with arrays and callbacks.

5️⃣ What are template literals?
Ans:Template literals are a way to write strings using backticks instead of single or double quotes.They allow variables and expressions to be included directly inside the string, making it more flexible and readable.They also support writing multi-line strings easily.
